# GITHUB Traffic data project with Django/Python and Celery

## Installation

```python
pip install -r requirements.txt
python manage.py makemigrations
python manage.py migrate
```

## Preview

```python
python manage.py runserver
```

## Test

```python
python manage.py test
```
